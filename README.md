# MSC Device OS

Plataforma de dispositivos inteligentes baseada em ESP32-S3, BLE Provisioning, OTA Runtime, Telemetria Distribuída e Edge AI.

## Objetivos

* Provisionamento BLE
* Atualização OTA
* Telemetria em tempo real
* Registro centralizado de dispositivos
* Fleet Management
* Edge Intelligence
* Mobile First
* Cloud Native

## Arquitetura

Device Layer

* ESP32-S3
* BLE Bootstrap
* OTA Engine
* Telemetry Engine
* Edge AI Runtime

Cloud Layer

* Device Registry
* OTA Server
* Telemetry API
* Fleet Manager

Mobile Layer

* Flutter
* BLE Provisioning
* Device Manager
* OTA Control
